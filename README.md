# Personalized Medicine Recommendation System

This project predicts heart disease risk using Machine Learning and provides personalized medicine and lifestyle recommendations.

## Features
- Heart Disease Prediction
- Risk Probability
- Personalized Medicine Recommendation
- Lifestyle Suggestions
- Interactive Streamlit UI

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Run Project

```bash
pip install -r requirements.txt
python -m streamlit run app.py
